# FutureIntern Internship – Prompt Engineering Task 01

## Repository Name
future_P_01

## Overview
This repository contains my first internship task at **Futureinton**, focused on **prompt engineering and AI-driven content creation**.

The goal of this task was to design a complete YouTube content package using AI tools — from prompt chains to final video output.

## What This Project Includes
- Prompt chains for generating YouTube content using AI
- A full YouTube video script created through prompt engineering
- AI-generated thumbnail concept and final thumbnail
- AI-generated YouTube video created from the script

## Tools & Platforms Used
- ChatGPT – Prompt engineering and script generation
- Pictory – Script-to-video AI generation
- Canva – Thumbnail design
- Bing Image Creator (DALL·E) – AI image generation
- GitHub – Version control and project submission

## Folder Structure
- **01_Prompt_Chain** – Prompt engineering documentation
- **02_Script** – Full YouTube video script
- **03_Thumbnail** – Thumbnail prompt and final image
- **04_Video** – Link to the generated YouTube-style video

## Learning Outcome
Through this task, I gained hands-on experience in:
- Prompt chaining
- Long-form AI content generation
- AI-assisted video creation
- Content packaging for digital platforms
